# 🎮 AluGames

Projeto desenvolvido com **HTML, CSS e JavaScript** durante o curso de **Lógica de Programação com JavaScript** da *Alura*.

O objetivo do projeto foi praticar lógica de programação, manipulação do DOM e interatividade em aplicações web através da criação de uma interface simples de aluguel de videogames.

---

## 📌 Sobre o projeto

O **AluGames** simula uma plataforma de aluguel de videogames, permitindo alterar dinamicamente o status dos jogos entre disponível e alugado.

Durante o desenvolvimento, foram aplicados conceitos fundamentais de JavaScript para controle de estados, manipulação de elementos HTML e interação com o usuário.

---

## 🚀 Funcionalidades

✔️ Exibição de jogos disponíveis

✔️ Alteração do status de aluguel dos jogos

✔️ Atualização dinâmica da interface

✔️ Mudança visual conforme disponibilidade

✔️ Interação em tempo real utilizando JavaScript

✔️ Interface simples e responsiva
